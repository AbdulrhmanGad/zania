Partner Ledger Report
 in MS Excel format (XLSX)

REPORT CONTENT

  • Header Fields: Company, Start Date, End Date, Print Date, The User who printed the report.
  • Document Body:
        Fields: Partner, Date, Journal, Reference, Label, Due date, Matching Number, Debit, Credit, Balance
        Group By: Partner
        Aggregate: Sum Debit, Sum Credit, Sum Balance
        Sort: Ascending Partner, Ascending Date
  • Report footer fields: Debit, Credit, Balance
  • Report Parameters: Start Date, End Date, Partners
  • Menu Full Path: Invoicing/Reporting/Partner Ledger (XLSX)

THIS MODULE REQUIRES THE INSTALLATION OF A MODULE:

"REPORT DESIGNER (XLSX, XLSM)"
